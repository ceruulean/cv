Prose Grinder Pandoc Templates

Contributor of PowerShell port.

Premade scripts to format text for traditional (book) publishing, using Pandoc’s templating features.

Formatted nonfiction and fiction manuscripts looks suspiciously similar to Markdown. Hence, in order to make the process less tedious for authors, we recommend writing in Markdown in the first place. Many authors use Windows, so a Windows-compatible port of the Prose Grinder utilties are needed.
